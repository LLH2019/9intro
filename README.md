# 9intro
plan9 操作系统介绍


这是一本plan9 介绍操作系统的开源图书。

由本人编写和翻译。

## 本书适合读者。

1. 想了解plan9 的 go语言开发者。
2. 需要有一定的Linux 使用基础。
3. 本书适合开发者、运维人员，不适合普通用户。

本书不涉及 plan9 内核讲解，只讲些使用方法，优缺点等。

讲解 plan9 的同时 也会与 Linux 与 unix做对比。受本人知识水平所限，难免遇到疏漏和 错误，希望读者指正。

## 写作目的

1. 督促自己学习
2. 推广plan9
3. 除了Linux/ unix 还有其他操作系统可以选择。
4. 给自己找点事干。
